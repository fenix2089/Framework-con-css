Sistema de grillas

El sistema de grilla funciona de una manera muy sencilla que se explica a continuacìon

 

Para llamar las columnas que uno desea solo tiene que iniciarse de esta forma,
se inicia un contenedor, luego la clase row para luego llamar las grillas,
se define el tamaño por ejemplo large-10 y luego se define el tamaño cuando es
responsive, por ejemplo
small-12
<div class="container">
<div class="row">
<div class="large-10 small-12"></div>
</div>
<div>

Puedes ver un ejemplo, copiando y pegando el codigo anterior y añadiendo este estilo a las grillas.
<style>
	div[class*="large"]:nth-child(2n+1){background-color: #F08080;}
	div[class*="large"]:nth-child(2n){background-color: #F08080;}
	div[class*="small"]:nth-child(2n+1){background-color: #F08080;}
	div[class*="small"]:nth-child(2n){background-color: #F08080;}
		@media (max-width: 768px) {
			[class*='small']:before {
		content: "small";
	}
}
@media (min-width: 768px) {
	[class*='small']:before {
		content: "large";
	}
}
</style>

Footers



Tenemos 3 tipos de footers, creados de forma sencilla pero diferentes.

Para mandar a llamar el primero el codìgo es:
<footer class="footer3"> </footer>
Para mandar a llamar el segundo con una posición fixed es:
<footer class="footer2"> </footer>
Para mandar a llamar el ultimo es con:
<footer class="footer3"> </footer>
