Bienvenido a Soopfc


Un framework diseñado para hacer sitios de forma fácil y rápida.


Aprenda como usar Soopfc en la creacion de su sitio web

Descarga

Esta es una versión estándar que viene con los archivos CSS y JavaScript minificados y no minificados. Esta opción requiere poca o ninguna configuración.
  • Descarga

Estructura del proyecto

Estructura del proyecto después de la descarga, extraiga los archivos en el directorio donde se encuentra su sitio web. Su directorio se verá algo así.

 

Misitioweb/
|--css/
|  |--soopfc.css
|--Font-awesone/
|--img/
|--js/
|  |--main.js
|  |--main2.js
|  |--jquery.fitvids.js
|  |--jquery-2.1.4.min.js
|  |--jquery.imagesloaded.min.js
|--index.html

Configuración de HTML

Configuración del html

A continuación, solo tiene que asegurarse de vincular los archivos correctamente en su página web. En general, es aconsejable importar archivos javascript al final al cuerpo de la pagina para reducir el tiempo de carga de la página. Siga el ejemplo a continuación sobre cómo importar Soopfc en su página web.

 

<!DOCTYPE html>
<html>
<head>
<link type="text/css" rel="stylesheet" href="css/soopfc.css"  media="screen,projection"/>
<meta name="viewport" content="width=device-width, initial-scale=1.0"/>
</head>
<body>
<script type="text/javascript" src="js/jquery-2.1.4.min.js"></script>
<script type="text/javascript" src="js/jquery.fitvids.js"></script>
<script type="text/javascript" src="js/jquery.imagesloaded.min.js"></script>
<script type="text/javascript" src="js/main.js"></script>
<script type="text/javascript" src="js/main2.js"></script>
</body>
</html>
